# SIT 2026

Landing page memorial da turma SIT 2026 do CEAP.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Estrutura de conteúdo

Os dados editáveis começam em `src/data/`. As seções completas serão adicionadas em `src/components/sections/` nas próximas etapas.
